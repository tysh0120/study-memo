デフォルトでコンパイルすると、保護機能が働くため、以下のオプションで無効にする

gcc -m32 -fno-stack-protector xxx.c

-m32:  32ビットでコンパイル
-fno-stack-protector: セキュリティ機能をオフにする


※ 本には-m32, -fno-stack-protectorだけの指定だったが、gdbでデバッグするため、-g オプションも付けた


-m32でコンパイルするには multilibの導入が必要
```bash
$ apt install gcc-multilib

# apt install g++-multilib  # C++も使うとき
```
