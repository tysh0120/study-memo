#asm #システムプログラミング 
### システムコール番号
/usr/include/asm/unistd_32.h  に定義

| 番号  | システムコール |
| --- | ------- |
| 1   | exit    |
| 3   | read    |
| 4   | write   |
### システムコールの呼び方
**EAX**にシステムコール番号
**EBX**に第１引数
**ECX**に第２引数
**EDX**に第３引数

を設定して
**int 0x80**　という特殊な命令を呼び出す

writeの呼び出し例
EAX: 0x4
EBX: ファイルでスクリプタ (標準出力： 0x1)
ECX: 出力バッファへのポインタ
EDX: 出力するバイト数
