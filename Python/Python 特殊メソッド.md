#python #プログラミング

ダブルアンダースコアで囲まれたメソッド
別名：マジックメソッド、ダンダー(**ダ**ブルア**ンダ**ースコア**メソッド**)

\_\_xxx\_\_() の形の特殊メソッドの場合、xxx(インスタンス名)で実行される

\_\_str\_\_(self): 文字列表現を定義
\_\_repr\_\_(self): オブジェクトの文字列表現を定義
\_\_call\_\_(self): インスタンス名()で呼ばれるメソッドを定義

### 比較(以下を定義することでソート可能オブジェクトに)
\_\_eq\_\_(self, other): == で比較された時の挙動を定義
\_\_lt\_\_(self, other): < で比較
\_\_gt\_\_(self, other): >
\_\_le\_\_(self, other): <=
\_\_ge\_\_(self, other): >=
