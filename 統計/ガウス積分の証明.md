$\int_{-\infty}^{\infty} e^{-x^2} dx$ 
定義域を2次元に拡張して $\int e^{-(x^2 + y^2)} dx$ の体積を考える

$\iint e^{-(x^2 + y^2)} dxdy$=  $\int {e^{-r^2}} dr$ = (シリンダーの体積)  = $\int 2{\pi}r e^{-{r^2}} dr$ = $[-{\pi}e^{-r^2}]^{\infty}_0$ = $\pi$

C := $\int_{-\infty}^{\infty} e^{-{x^2}} dx$  とおくと

$\iint {e^{-(x^2+y^2)}}dx$ = $\int{C e^{-y^2}}dy$ = $C^2$ = $\pi$

C = $\sqrt{\pi}$
