### 1. 行を削除
```
df.dropna(subset=['col1'], inplace=True)
# subsetで列を指定しないと何れかの列に対してNone値を含む行すべてを削除
```
### 2. 列を削除
```
df.drop('col1', axis=1, inplace=True)
```
### 3. 中間値などで補完
```
median = df['col1'].median()
df['col1'].fillna(median)
```
